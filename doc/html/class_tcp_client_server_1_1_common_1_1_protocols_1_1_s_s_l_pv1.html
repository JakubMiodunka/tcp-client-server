Detailed Description

Handler for data manipulation related to Simple Session Layer Protocol version 1 - simple 5 layer protocol created by Jakub Miodunka for sake of 'chat-net' project.

Sum of bytes contained by packet header is equal to length of its payload. Currenty it is prefered to use version 2 as it is much more efficient.
